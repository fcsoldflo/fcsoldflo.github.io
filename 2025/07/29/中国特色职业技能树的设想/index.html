    ###关于开发适合中国大学生体质的技能树的设想（大创项目）

    一、项目背景与意义（立项理由）

    当前，中国大学生面临的职业选择日益多样，然而多数学生在职业规划方面缺乏有效指导。学校的就业服务多停留在宣讲层面，市面上的求职平台大多面向职场人士，缺乏贴合中国高校专业与学生成长路径的系统性技能树指导工具。本项目旨在开发一个面向中国大学生的职业技能树网站，将专业、兴趣、技能、岗位要求相结合，构建清晰可视化路径，帮助学生找到适合自己的职业方向并规划成长路线，提升就业精准度和主动性。

    🧭 产品功能拆解（面向未来）

    功能模块 目标说明 技术建议
    🎮 技能树展示 树状展示每个职业的技能路径 React + React Flow / D3.js
    📝 技能节点编辑 点技能可编辑介绍/资源/经验 表单 + Modal 组件（移动端友好）
    📦 数据存储 数据能读写、版本管理 JSON 存储（初期）→ GitHub 仓库 / 后端数据库
    🌍 响应式设计 在手机/PC都能正常展示 Tailwind CSS + Flex/Grid布局
    🧑‍🤝‍🧑 共同编辑 所有人都能参与贡献 GitHub协作 or 简易用户系统（后期）
    🌐 开源部署 网页对所有人开放访问 Vercel / GitHub Pages 部署网站
